## Revision Note

The previous 5-family version did not include Pharmaceuticals as a standalone product family. This version adds:

- `Pharmaceuticals` to `PRODUCT_FAMILIES`
- `Pharmaceuticals` to `seasonal_index.csv`
- `Pharmaceuticals` rows in `annual_demand_by_region_product.csv`
- `Pharmaceuticals` rows in `monthly_demand_by_region_product.csv`
- `commodity_product_family_mapping.csv` for auditability

Because the uploaded O/D source data contains generic labels such as `품목06` rather than full commodity descriptions, the mapping remains heuristic. In this revision, `품목06` is mapped to `Pharmaceuticals`; `품목07` and `품목08` are also reserved for Pharmaceuticals in the CLI config if they appear in future O/D files. If the official commodity-code dictionary or Group C classifier is provided, update `engine/config.py` and rerun the CLI.

## How to Run A5 CLI

Place the three source files in a local `data/` folder:

- `od_clean_long_2023.csv`
- `warehouse_geocoded.csv`
- `warehouse_capacity_17_regions.csv`

Then run from the project root:

```bash
python -m engine.cli build-od --input data/ --output output/
```
